## 📋 Overview

**FANTASTIC-4 × ODOO** is a comprehensive Human Resource Management System designed to streamline and modernize workforce operations. Built as a custom Odoo integration with a sleek modern frontend, it covers everything from employee onboarding to payroll, attendance, and analytics.

## ✨ Core Modules

| Module | Description |
|---|---|
| 👤 **Employee Management** | Profiles, departments, roles, org charts |
| 🕐 **Attendance & Leave** | Clock-in/out, leave requests, approvals |
| 💰 **Payroll** | Salary structures, payslips, tax deductions |
| 🎯 **Recruitment** | Job postings, applicant tracking, onboarding |
| 📊 **Performance** | KPIs, appraisals, goal tracking |
| 📈 **Analytics** | Dashboards, reports, workforce insights |

## 🛠️ Tech Stack

- **Backend**: [Odoo 17](https://www.odoo.com/) (Python, PostgreSQL)
- **Frontend**: React + Vite (custom UI)
- **API**: Odoo JSON-RPC / REST API
- **Auth**: Odoo session-based authentication
